## Vue Cli
แบบที่ 1 ถ้าใช้ vue-cli
```bash
$ git clone git@github.com:yuttasakcom/vue-component.git && cd vue-component/01.vue-cli
$ yarn install or npm install
$ yarn dev or npm run dev
```
